# Exploratory-Data-Analysis-in-SQL
This project focuses on performing Exploratory Data Analysis (EDA) on a global layoffs dataset using SQL. The objective was to analyze layoffs across different companies, industries, countries, and time periods to identify patterns and trends in workforce reductions.After cleaning the dataset, SQL queries were used to aggregate and analyze the data to uncover insights about layoffs worldwide.

# Tools Used
* SQL (MySQL)
* GitHub

# Analysis Performed
* Calculated total layoffs by company
* Analyzed layoffs by industry and country
* Examined yearly and monthly layoff trends
* Identified companies with the highest layoffs
* Explored layoffs by funding stage and percentage laid off

# SQL Concepts Used
* GROUP BY
*ORDER BY
*Aggregate functions (SUM, MAX, MIN)
*CTE
*Window functions

# Outcome
The analysis helped identify key trends in global layoffs, including the industries and companies most affected and how layoffs changed over time.
